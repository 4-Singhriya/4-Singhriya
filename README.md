## 🚀 Major Projects  

### 🛰️ **Autonomous Drone Detection System**  
A **Multimodal Camera-Based Drone Detection System** designed to identify unauthorized drone activities in restricted areas.  
The system leverages **YOLO, RetinaNet (Detectron2), and One-Class SVM** to detect drones in images, videos, and live feeds — including mobile camera streams via **DroidCam**.  

- 🧠 **Core Tech:** Python, OpenCV, TensorFlow, Detectron2, One-Class SVM  
- 🔍 **Key Features:** Real-time detection | Multi-model pipeline | Mobile integration  
- 🧩 **Focus:** Vision AI, ML model fusion, and applied deep learning  
- 🔗 **Repo:** [Autonomous Drone Detection System](https://github.com/4-Singhriya/Autonomous-Drone-Detection-System)

---

### 🔍 **LexiVerse — Browser-based Search Engine**  
A **high-performance, browser-native search engine** capable of querying **over a million entries in milliseconds**, proving you don’t need massive backends for massive data.  

- ⚡ **Frontend:** HTML5, CSS3, JavaScript  
- 📊 **Data Handling:** JSON, Array, Map  
- 🧮 **Algorithms:** Custom relevance sort | Frequency counting | Search ranking  
- 🧩 **Focus:** Data structures, algorithmic optimization, and frontend scalability  
- 🔗 **Repo:** [LexiVerse](https://github.com/4-Singhriya/LexiVerse)

### 🎬 **Movie Genre Mixer Chatbot**  
An **AI-driven movie genre recommender** that creatively blends genres to generate unique film ideas and suggestions.  
Built using **Google Gemini API**, **TMDB API**, and a **CustomTkinter GUI**, it interacts naturally with users to suggest hybrid genres and movie concepts.  

- 🤖 **Core Tech:** Python, Gemini API, TMDB API, CustomTkinter  
- 💬 **Key Features:** Conversational interface | Dynamic genre mixing | Real-time data from TMDB  
- 🧠 **Focus:** Generative AI, API integration, and creative recommendation systems  
- 🔗 **Repo:** [Movie Genre Mixer Chatbot](https://github.com/4-Singhriya/Movie-Genre-Mixer-Chatbot)
